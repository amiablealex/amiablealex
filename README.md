Manufacturing Engineer in the UK. At work: a  mix of project management and digital transformation / data science. At home: developing small Flask apps on a Raspberry Pi for friends and family.

### What I build:
- Manufacturing data & BI (Fabric, Power BI, DAX, KPI modelling)
- Self-hosted web apps and home infrastructure (Flask, Pi, Home Assistant)
- Production-leaning hobby projects: local dev on the Pi, ephemeral previews on Railway, zero-downtime deploys to the things people actually use. 

### Featured Projects:
- KitSniff - A daily ingredient guessing game. Guess the mystery ingredient by its colour, flavour, food group, and type. New puzzle every day, everyone gets the same one.
- F1 Predictions - A predictions league game for a friend group. Predict the top 10, top 3 in qualifying, pole time, fastest lap etc. plus rotating per-round bonuses that change every weekend.
- Chess Storyteller - Transform a PGN file into calm, descriptive narratives. No algebraic notation, just rich positional language.
- Vantix - An FPL league dashboard with real-time stats, extended insights, interactive charts, and automated data updates.

### How I Work:
I lean heavily on AI coding assistants (Claude, Gemini) as a force multiplier. The engineering judgement - system design, user experience, debugging when things go wrong in production, deciding what's worth implementing - comes from an engineering data science background and university CS fundamentals. The output is more code than I'd write unassisted; the quality bar is mine.
I enjoy taking deployment seriously - the bit between "it works for me" and "it's being used by real users without me having to think about it." KitSniff has had daily active users since May 2025 without a missed day.

### Tech / Tools:
- Stack: Mostly Python (Flask) with Postgres, DAX/M for the BI work, YAML for the home infra side.
- Infrastructure: Raspberry Pi for local dev and always-on lightweight stuff, Railway for production deployment ephemeral environments and managed Postgres, Hetzner Cloud for VPS workloads that need to be more serious, Cloudflare for tunnels and DNS. Resend for transactional email.
